\doxysubsection{Detailed Description}
Base class for snakes. It specializes into \mbox{\hyperlink{classDangerousSnake}{Dangerous\+Snake}} and \mbox{\hyperlink{classNonDangerousSnake}{Non\+Dangerous\+Snake}}. It is derived from class \mbox{\hyperlink{classAnimal}{Animal}} and add a boolean Snake\+::dangerous to specify if a type of snake is dangerous or not.
